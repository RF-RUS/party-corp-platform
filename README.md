# Party Corp - E-commerce Optimization Platform

AI-powered e-commerce optimization platform for Shopify and WooCommerce stores.

## Features

- **User Authentication** - Register and login with email/password
- **Store Connection** - Connect Shopify or WooCommerce stores
- **AI Recommendations** - Smart recommendations to grow your sales
- **Analytics Dashboard** - View revenue, orders, AOV, and conversion rates

## Tech Stack

- Next.js 14
- TypeScript
- In-memory database (MVP)

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
